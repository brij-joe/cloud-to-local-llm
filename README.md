This project offers a practical, step‑by‑step guide to running large language models (LLMs) locally. By combining Transformers, Hugging Face, LangChain, and LangGraph, you can:
* Download and run LLMs directly on your machine (starting with smaller models for easier setup).
* Integrate them into pipelines for structured tasks and workflows.
* Build graph‑based applications that manage complex interactions.

With this approach, you gain full control over your LLM environment without relying on external APIs.

**System requirements:** A multi‑core CPU and a GPU are essential for local execution. For NVIDIA GPUs, install CUDA along with the appropriate Torch packages to ensure compatibility.
