This project provides a step-by-step guide to running large language models (LLMs) locally. By combining **Hugging Face**, **LangChain**, and **LangGraph**, you can:

- Download and execute LLMs on your own machine (starting with smaller models for ease of use).
- Integrate them into pipelines for structured tasks and workflows.
- Design graph-based applications that handle complex interactions.

With this setup, you gain complete control over your LLM environment without depending on external APIs.
